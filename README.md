# RustERP

A small and custom ERP system for small company.

## Getting Started

Clone the repository and simply run:

```
cargo run
```

If you want to test you can run:

```
cargo test
```

### Prerequisites

None at this time!


### Installing

The easiest way to install the software is basically running:

```
cargo install
```
